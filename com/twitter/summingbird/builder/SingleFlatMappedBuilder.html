SingleFlatMappedBuilder

class SingleFlatMappedBuilder[Event, Time, Key, Value] extends FlatMappedBuilder[Time, Key, Value]

SingleFlatMappedBuilder is the abstract class used to transform an EventSource into a FlatMappedBuilder. These guys can be fed into MergedFlatMappedBuilder instances to combine with other event sources.
